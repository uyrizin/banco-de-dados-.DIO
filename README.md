# Projeto Banco de Dados - DIO

Este projeto foi desenvolvido como parte dos cursos da **Digital Innovation One (DIO)**.  
O objetivo é praticar conceitos de **modelagem de dados**, **SQL** e **administração de banco de dados**.

## 🚀 Tecnologias utilizadas
- SQL Server (ou outro banco relacional)
- Scripts SQL (DDL, DML, DQL)
- Ferramentas de modelagem (ex: MySQL Workbench, SQL Server Management Studio)
